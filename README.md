## 📌 实用说明
### 🔍 快速使用技巧
1. 按 `T` 键唤醒仓库搜索框，输入关键词（如「汽车理论」「PID」）秒找文件；
2. 点击文件名链接可直接查看/下载文件，部分文件支持在线预览；
3. 文件夹后标注的「最近更新」为内容维护时间，优先看最新更新的资料。

### 📝 贡献须知
1. 本仓库为**西南交大2026届车辆工程专业**学习资料备份，仅限学习交流，禁止商用；
2. 如有文件错误/补充需求，可提交 PR 或在 Issues 反馈；
3. 新增文件建议按「文件名.格式」命名（如「汽车理论真题.md」），保持格式统一。
